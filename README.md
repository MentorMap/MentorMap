# MentorMap
Website development project for MentorMap

## Running the App
To clone to your local machine:
```
$ git clone
$ cd MentorMap
```
Install dependencies:
```
$ npm install
```
To run the startup script (in debug mode)
```
$ DEBUG=mentormap:* npm start
```
or
```
$ gulp nodemon
```
Navigate to ```localhost:3000```
